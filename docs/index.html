MediaNameFixer

专为摄影师和影像爱好者设计的文件批量重命名工具

自动读取照片/视频的拍摄时间（EXIF、QuickTime 元数据），按时间规则一键重命名， 彻底解决相机默认文件名混乱、难以检索的问题。

功能特色
📷
EXIF 智能解析
自动读取 JPEG、RAW 等照片文件的 EXIF 元数据，精准提取拍摄时间作为重命名依据。支持佳能、尼康、索尼、富士等主流相机格式，无需手动输入。
🎬
视频时间戳支持
深度解析 MP4、MOV、3GP 等视频文件的 QuickTime / mvhd 元数据，正确处理 creation_time 与时区偏移，让视频文件名也能精准反映拍摄时刻。
📁
一键归档整理
支持按年/月/日自动创建文件夹结构，将重命名后的文件有序归档。告别杂乱的根目录，项目资料一目了然，后期查找高效便捷。
🖼️
缩略图预览
内置高性能缩略图生成引擎，文件列表中即可快速预览照片与视频内容。支持并发加载与懒加载优化，万级文件也能流畅浏览。
🔄
自动更新
启动时自动检测最新版本，支持一键下载并安装更新。双地址检查机制（官网 + GitHub Releases）确保获取通道稳定可靠。
💻
离线运行
基于 Tauri 构建的本地桌面应用，所有文件处理均在本地完成，无需联网上传。数据隐私完全由你掌控，随时随地安心使用。
